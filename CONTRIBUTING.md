# Contributing to NuroStack

We love your input! We want to make contributing to NuroStack as easy and transparent as possible.

## Pull Request Process

1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the existing style and typography.
3. Update the README.md with details of changes if applicable.
4. Submit a Pull Request with a clear description of the improvements.

## Development Setup

The project is a monorepo:
- `frontend/`: React + Vite application.
- `automation/`: Node.js fetching scripts.
